Coding Python for transaltion to Javascript

Some constraints when coding python so that it can be reused as Javascript

Things you can't use

The % operator for string formatting

There is no sprintf equivalent in JS runtime

Asigniment to a tuple a,b = c,d

Things which do convert

  • Slices of strings and arrays
  • exceptions (with a hack)
  • for i in ... (generates an iterator)
  • mystring.encode('utf-8') (with a hack only for utf-8)
